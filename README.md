# 📊 Election Data Analysis (2019)

## 📌 Overview
This project presents an analytical study of the Indian General Election data 2019 using Statistical Analysis, Tableau Visualizations, and SQL (Google BigQuery).

The project focuses on evaluating voting trends, party performance, candidate competitiveness, and electoral patterns across selected Indian states through data-driven insights and interactive visualizations.

---

## 🎯 Objectives
- Analyze state-wise voting trends
- Compare political party performance
- Evaluate winner and runner-up vote margins
- Identify electoral competition patterns
- Generate meaningful insights using data visualization techniques

---

## 🗺️ States Analyzed
- Maharashtra
- Rajasthan
- Gujarat
- Uttar Pradesh
- Karnataka
